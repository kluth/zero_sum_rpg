# WebApp

Dit project is gegenereerd met [Angular CLI](https://github.com/angular/angular-cli) versie 17.3.17.

## Development server

Draai `ng serve` voor een dev server. Navigeer naar `http://localhost:4200/`. De application zal automatisch opnieuw laden als je een van de source files wijzigt.

## Code scaffolding

Draai `ng generate component component-name` om een nieuwe component te genereren. Je kunt ook `ng generate directive|pipe|service|class|guard|interface|enum|module` gebruiken.

## Build

Draai `ng build` om het project te builden. De build artifacts worden opgeslagen in de `dist/` directory.

## Running unit tests

Draai `ng test` om de unit tests uit te voeren via [Karma](https://karma-runner.github.io).

## Running end-to-end tests

Draai `ng e2e` om de end-to-end tests uit te voeren via een platform naar keuze. Om dit command te gebruiken, moet je eerst een package toevoegen die end-to-end testing capabilities implementeert.

## Further help

Om meer hulp te krijgen over de Angular CLI gebruik `ng help` of bekijk de [Angular CLI Overview and Command Reference](https://angular.io/cli) pagina.
