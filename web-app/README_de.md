# WebApp

Dieses Projekt wurde mit der [Angular CLI](https://github.com/angular/angular-cli) Version 17.3.17 generiert.

## Development server

Führe `ng serve` für einen Dev Server aus. Navigiere zu `http://localhost:4200/`. Die Application wird automatisch neu laden, wenn du eine der Source Files änderst.

## Code scaffolding

Führe `ng generate component component-name` aus, um eine neue Component zu generieren. Du kannst auch `ng generate directive|pipe|service|class|guard|interface|enum|module` verwenden.

## Build

Führe `ng build` aus, um das Projekt zu builden. Die Build Artifacts werden im Verzeichnis `dist/` gespeichert.

## Running unit tests

Führe `ng test` aus, um die Unit Tests via [Karma](https://karma-runner.github.io) auszuführen.

## Running end-to-end tests

Führe `ng e2e` aus, um die End-to-End Tests über eine Plattform deiner Wahl auszuführen. Um dieses Command zu nutzen, musst du zuerst ein Package hinzufügen, das End-to-End Testing Capabilities implementiert.

## Further help

Um mehr Hilfe zur Angular CLI zu erhalten, nutze `ng help` oder schau dir die [Angular CLI Overview and Command Reference](https://angular.io/cli) Seite an.
